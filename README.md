# Smart Pharmaceutical Pillbox

A smart pharmaceutical organizer developed using Arduino that helps users take medication on time by sending SMS notifications and displaying LED alerts.

## Project Description

This project was developed as a STEM research project. The device helps elderly people or patients with chronic diseases manage their medication schedule more easily.

The system uses an Arduino board and a SIM module to send automatic SMS notifications reminding the user when it is time to take their medication.

LED indicators show the corresponding day of the week for medication administration.

## Features

- SMS reminders for medication
- LED notifications for daily doses
- Compact 3D printed case
- Arduino-based control system

## Technologies Used

- C++
- Arduino
- SIM800L GSM module
- PTC Creo Parametrics (3D design)
- 3D printing (PLA)

## Hardware Components

- Arduino UNO R3
- SIM800L GSM module
- LEDs
- Resistors
- Breadboard
- Jumper wires

## How It Works

The Arduino program controls the notification system. At predefined times, the GSM module sends an SMS reminder to the user's phone. LEDs light up to indicate the correct day for medication.
